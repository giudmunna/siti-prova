Per usare la versione SQL locale:
1. Avvia XAMPP e assicurati che MySQL sia attivo.
2. Apri http://localhost/sirio/install_db.php per creare il database "sirio" e importare i corsi iniziali.
3. Vai su http://localhost/sirio/admin/ per accedere con admin / sirio2026.
4. Usa la sezione Corsi per aggiungere o modificare corsi e orari, e Prenotazioni per vedere le richieste di prova.
